# Esbuild Template
> My boilerplate configuration for js projects

### Setup
```bash
$ npx degit ale-cci/esbuild-template <PROJECT-NAME>
$ cd <PROJECT-NAME>
$ ./configure
$ git init
$ git config core.hooksPath '.git-hooks'
```

### About
- Unittest framework: `jest`
- Bundler: `esbuild`
- Linter: `eslint`
